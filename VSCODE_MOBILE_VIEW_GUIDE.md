# 📱 VS Code 查看手机版样式 - 完整指南

## 🎯 最佳推荐：Browser Preview

### 安装步骤：

1. **打开 VS Code**
2. **点击左侧扩展图标**（或按 `Cmd+Shift+X` / `Ctrl+Shift+X`）
3. **搜索**：`Browser Preview`
4. **安装**：点击 "Install"（作者：auchenberg）

### 使用方法：

1. **打开您的 HTML 文件**（如 `index.html`）
2. **按 `Cmd+Shift+P` (Mac) 或 `Ctrl+Shift+P` (Windows)**
3. **输入**：`Browser Preview: Open Preview`
4. **选择**：`Browser Preview: Open Preview`
5. **在预览窗口中**：
   - 按 `F12` 打开开发者工具
   - 或点击右上角的设置图标
   - 点击**设备模拟图标**（手机/平板图标）
   - 或按 `Cmd+Shift+M` (Mac) / `Ctrl+Shift+M` (Windows)
6. **选择设备**：
   - iPhone 12/13/14/15
   - iPhone SE
   - iPad
   - Samsung Galaxy
   - 或自定义尺寸

### 优点：
- ✅ 在 VS Code 内直接预览
- ✅ 支持 Chrome DevTools
- ✅ 可以切换不同设备
- ✅ 实时更新

## 🚀 方案 2：Live Preview（Microsoft 官方）

### 安装步骤：

1. **搜索**：`Live Preview`
2. **安装**：Microsoft 官方扩展

### 使用方法：

1. **右键 HTML 文件** → "Show Preview"
2. **在预览窗口右上角**点击设备图标
3. **选择设备尺寸**

### 优点：
- ✅ Microsoft 官方扩展
- ✅ 简单易用
- ✅ 内置响应式预览

---

## 📐 方案 3：Responsive Viewer（多设备对比）

### 安装步骤：

1. **搜索**：`Responsive Viewer`
2. **安装**：作者 peterhdd

### 使用方法：

1. **打开 HTML 文件**
2. **按 `Cmd+Shift+P`**
3. **输入**：`Responsive Viewer: Open`
4. **选择设备组合**或自定义尺寸

### 优点：
- ✅ 同时查看多个设备
- ✅ 方便对比不同屏幕
- ✅ 支持自定义尺寸

---

## 💡 最简单的方法（无需安装扩展）

### 使用 Chrome DevTools：

1. **在浏览器中打开网站**
   - 可以直接双击 `index.html` 打开
   - 或使用 Live Server

2. **按 `F12` 打开开发者工具**

3. **点击设备模拟图标**
   - 在开发者工具左上角
   - 或按 `Cmd+Shift+M` (Mac) / `Ctrl+Shift+M` (Windows)

4. **选择设备**：
   - iPhone 12 Pro (390x844)
   - iPhone SE (375x667)
   - iPad (768x1024)
   - 或自定义尺寸

5. **实时查看移动端样式**

## 🎯 推荐使用流程

### 最推荐：Browser Preview

1. **安装 Browser Preview 扩展**
2. **打开 `index.html`**
3. **按 `Cmd+Shift+P` → "Browser Preview: Open Preview"**
4. **在预览中按 `F12`**
5. **点击设备模拟图标**
6. **选择 iPhone 或其他设备**

这样您就可以在 VS Code 中直接看到手机版的样式了！
