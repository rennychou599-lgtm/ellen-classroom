## 如何更換插圖？

### 方法 1: 使用 unDraw 其他插圖

1. 訪問 https://undraw.co/illustrations
2. 搜索你想要的主題（例如：study, learning, education）
3. 找到喜歡的插圖後，點擊下載
4. 記下插圖的檔案名稱（例如：`studying.svg`）
5. 在 `index-ellen.html` 中替換圖片網址：

```html
<!-- 原本的 -->
<img src="https://raw.githubusercontent.com/unDraw/illustrations/main/illustrations/online-learning.svg">

<!-- 改成 -->
<img src="https://raw.githubusercontent.com/unDraw/illustrations/main/illustrations/studying.svg">
```

### 方法 2: 使用本地圖片

1. 在專案資料夾中創建 `images` 資料夾
2. 將你的圖片放入資料夾中
3. 修改 HTML 中的圖片路徑：

```html
<img src="images/your-image.svg" alt="你的圖片描述">
```

## 插圖顏色自訂

unDraw 的插圖支援顏色自訂：

1. 在 unDraw 網站上選擇插圖
2. 點擊右上角的調色板圖標
3. 選擇你想要的主題色（建議使用網站的紫色 #667eea）
4. 下載並使用新顏色的插圖

## 動畫效果說明

### 漂浮動畫 (Float)
- 用於主要插圖（Hero 和關於區）
- 上下緩慢移動，營造輕盈感
- 動畫週期：3秒

### Hover 放大效果
- 用於小型插圖（學習心得、複習進度）
- 滑鼠移上去時稍微放大
- 增加互動感

## 響應式設計

插圖會根據螢幕尺寸自動調整：

- **桌面 (>1024px)**: 完整尺寸顯示
- **平板 (768-1024px)**: 縮小至中等尺寸
- **手機 (<768px)**: 縮小至適合手機的尺寸

## 性能優化建議

1. **使用 SVG 格式**: 
   - 檔案小，載入快
   - 可無限縮放
   - 支援 CSS 樣式

2. **延遲載入** (如果圖片很多):
   ```html
   <img src="..." loading="lazy" alt="...">
   ```

3. **壓縮圖片** (如果使用 PNG/JPG):
   - 使用 TinyPNG 或 ImageOptim
   - 建議檔案大小 < 100KB

## 常見問題

**Q: 插圖顯示不出來？**
A: 檢查網路連線，確保可以訪問 GitHub。或者下載插圖到本地使用。

**Q: 想要不同風格的插圖？**
A: 可以嘗試其他插圖庫，或者使用 AI 生成工具（如 Midjourney、DALL-E）。

**Q: 如何移除動畫效果？**
A: 在 `styles-ellen.css` 中移除或註釋掉 `animation` 屬性：
```css
.hero-illustration {
    /* animation: float 3s ease-in-out infinite; */
}
```

**Q: 插圖太大或太小？**
A: 調整 CSS 中的 `max-width` 值：
```css
.hero-illustration {
    max-width: 600px; /* 調整這個數值 */
}
```
